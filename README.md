# Code Sync Admin Panel

[![Node.js](https://img.shields.io/badge/Node.js-16.x-blue)]()

This is the **Admin Panel** for the Code Sync Application. It allows the admin to monitor users, view submitted code, and manage application data.

---

## Features

- View registered users
- View submitted/generated code
- Monitor code activity
- Simple and professional web interface
- API integration with Node.js server

---

## Tech Stack

- **Node.js**
- **Express.js**
- **EJS / HTML / CSS / JS** (or your preferred front-end template engine)
- **Axios / Fetch** (for API calls to the server)
