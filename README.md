# Code Sync Admin Panel

[![Node.js](https://img.shields.io/badge/Node.js-16.x-blue)]()

This is the **Admin Panel** for the **Code Sync Application**. It allows the admin to manage users, monitor code submissions, and maintain the overall application activity. The admin panel is built using **Node.js** and communicates with the **Node.js backend server**.

---

## Features

- ✅ Admin login and authentication  
- ✅ View all registered users  
- ✅ View submitted/generated code from users  
- ✅ Monitor and manage code submissions  
- ✅ Clean and professional dashboard UI  
- ✅ API integration with the Node.js server  

---

## Tech Stack

- **Node.js** – Backend runtime  
- **Express.js** – Web framework  
- **EJS / HTML / CSS / JS** – Frontend templates  
- **Axios / Fetch** – For communicating with server APIs  
- **MongoDB** – Database (via server)
